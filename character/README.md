# Character Query Task

## Objective
Write GraphQL queries to fetch specific characters from the Rick & Morty API using their IDs.

## Endpoint
https://rickandmortyapi.com/graphql

## Requirements
For each character ID (1, 2, 3, 4), create:
- A .graphql file with the query
- A .json file with the API response

## Fields included:
- id
- name
- status
- species
- type
- gender

## Files
- character-id-1.graphql
- character-id-1-output.json
- character-id-2.graphql
- character-id-2-output.json
- character-id-3.graphql
- character-id-3-output.json
- character-id-4.graphql
- character-id-4-output.json
